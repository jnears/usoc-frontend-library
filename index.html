Lists

Unordered Lists

Unordered lists, or bulleted lists, are used when a set of items can be placed in any order. An example is a shopping list:

  • bread
  • coffee
  • milk
  • butter

Ordered Lists

Ordered lists, or numbered lists, are used to display a list of items that need to be placed in a specific order. An example would be cooking instructions, which must be completed in order for the recipe to work:

  1. List item 1
  2. List item 2
  3. List item 3
  4. List item 4
  5. List item 5
    1. Typography
    2. Grids
  6. List item 6

Definition

Description lists associate specific names and their values within a list, for example items in an ingredient list and their descriptions, article metadata such as authors and categories and their values, or competition winners and the years in which they won.
